# Coding challenge

## Features

- Enable/Desable Plugins
- Navigate between tabs

# Pre-requisites

- Install [Node.js](https://nodejs.org/en/)
- install project packages by running

```bash
  npm install
```

OR

```bash
  yarn install
```

## Running Program

To run program, run the following command.

```bash
  npm start
```

OR

```bash
  yarn start
```
